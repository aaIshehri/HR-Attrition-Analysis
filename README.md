# 📊 HR Attrition Analysis

## 📌 Project Overview

This project analyzes employee attrition using HR data to identify key factors influencing employee turnover.
The goal is to provide actionable insights that help organizations improve employee retention.

---

## 🎯 Objective

To understand the main drivers of employee attrition and suggest data-driven recommendations to reduce turnover.

---

## ❓ Key Questions

* Which employees are most likely to leave?
* Does job satisfaction affect attrition?
* How do age and job level impact employee turnover?
* Are certain departments or education fields more affected?
* What role does workload (overtime) play?

## 🔍 Key Insights

* Employees at **Job Level 1** have the highest attrition rate.
* The age group **25–34** shows the highest turnover.
* **R&D department** has the highest number of resignations.
* Employees with lower job satisfaction are more likely to leave.
* Certain fields like **Life Sciences** show higher attrition.

---

## 💡 Recommendations

* Improve onboarding experience for new employees.
* Review salary structure for entry-level roles.
* Reduce excessive overtime and workload pressure.
* Focus on employee engagement programs.
* Investigate high-turnover departments for deeper issues.

---

## 🛠 Tools Used

* Power BI
* Excel

---

## 📁 Dataset

The dataset contains 1470 employee records with features such as age, job role, salary, satisfaction, and attrition status.

---

## 📈 Outcome

This project demonstrates how data analysis can be used to uncover hidden patterns in employee behavior and support better HR decision-making.
